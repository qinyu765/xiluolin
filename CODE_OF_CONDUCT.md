# 社区行为准则

XiLuoLin 致力于建立友好、专业、包容且以解决问题为导向的开源社区。

## 我们鼓励

- 尊重不同经验、背景、观点和表达方式。
- 针对问题、代码和设计进行具体、可验证的讨论。
- 善意解释，主动澄清误解，并为新贡献者提供必要上下文。
- 接受建设性反馈，对自己的影响负责。
- 保护用户、贡献者和第三方的隐私与安全。

## 不可接受的行为

- 骚扰、侮辱、歧视、威胁或人身攻击。
- 发布他人的私人信息、密钥、录音或未授权内容。
- 持续打断讨论、恶意消耗维护资源或操纵协作流程。
- 任何在专业协作环境中明显不合适的行为。

## 执行

维护者可以编辑、隐藏或删除违反本准则的 Issue、评论、PR 和其他内容，并根据行为严重程度采取警告、临时限制或永久禁止参与等措施。

如果你需要报告行为问题，请优先通过仓库维护者可见的私密渠道联系；如果暂时没有可用私密渠道，可创建不包含敏感细节的 Issue，请求维护者建立私下联系。
