### 1. 新增任务（T012-T022）

| 任务 ID | 任务名称 | 优先级 | 说明 |
|---|---|---|---|
| T012 | 实现录音模块 | P0 | 麦克风录音、音频文件生成、时长统计、状态通知 |
| T013 | 实现全局快捷键模块 | P0 | 注册全局快捷键，支持长按和切换两种录音模式 |
| T014 | 实现输出模块（复制和自动粘贴） | P1 | 复制到剪贴板、模拟粘贴、失败兜底 |
| T015 | 实现主界面 | P0 | 人格选择、录音按钮、状态展示、文本展示、复制按钮 |
| T016 | 实现历史页 | P1 | 展示最近输入历史，支持复制历史结果 |
| T017 | 实现人格页 | P1 | 内置人格、自定义人格管理、设置默认人格 |
| T018 | 实现热词页 | P1 | 热词列表、新增、编辑、启用/停用 |
| T019 | 实现统计页或首页卡片 | P1 | 协作次数、口述时间、生成字数、节省时间、常用人格 |
| T020 | 实现设置页 | P0 | ASR 配置、LLM 配置、快捷键、录音模式、输出方式 |
| T021 | 实现错误处理和兜底 | P0 | 覆盖各类失败场景，确保有明确提示和兜底方案 |
| T022 | 端到端验证和调优 | P0 | 完整闭环验证，确保 MVP 可演示、可解释、可复现 |
