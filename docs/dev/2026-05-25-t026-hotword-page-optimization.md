## 为什么这么做

### 功能定位

1. **热词的作用**:
   - 帮助 AI 理解用户的专有词汇
   - 减少语音识别的误识别
   - 提升文本整理的准确性

2. **热词的使用场景**:
   - 专有名词:项目名、产品名、人名、地名
   - 技术词汇:框架名、库名、API 名称
   - 行业术语:特定领域的专业词汇
   - 个人习惯:常用缩写、昵称

3. **热词的实现方式**:
   - 启用的热词作为上下文注入到文本整理 prompt
   - 格式:"用户常用词汇: Next.js(框架), XiLuoLin(项目名)..."
   - AI 在整理文本时会参考这些词汇

### UI 设计

1. **说明文案**:
   - 简洁明了,一句话说明热词的作用
   - 列举常见使用场景,帮助用户理解
   - 说明热词如何生效

2. **字段说明**:
   - 每个字段添加说明和示例
   - 降低用户理解成本
   - 提升新建热词的成功率

3. **上下文展示**:
   - 让用户看到热词如何注入到 prompt
   - 增强用户对功能的理解
   - 提升用户信任感
