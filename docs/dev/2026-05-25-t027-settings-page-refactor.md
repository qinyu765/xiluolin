## 实际改动

### 前端改动 (src/main.tsx)

1. **从主界面移除配置卡片**:
   - 删除"文本整理服务"卡片(T007 OpenAI 整理)
   - 删除"语音识别服务"卡片(T006 智谱 ASR)
   - 这些配置移到设置页

2. **新增设置页分类 Tab**:
   - 使用 Tabs 组件实现顶部水平 Tab
   - Tab 项:通用、模型配置

3. **通用设置 Tab**:
   - 快捷键配置:长按模式快捷键、切换模式快捷键

   - 自动保存历史(开关)

4. **模型配置 Tab**:
   - 智谱 GLM-ASR-2512 配置:
     - API Key(密码输入框)
     - Base URL
     - 模型名
     - 保存按钮
   - OpenAI Responses API 配置:
     - API Key(密码输入框)
     - Base URL
     - 模型名
     - 保存按钮

5. **优化设置页布局**:
   - 使用卡片组织相关设置项
   - 每个设置项添加说明文字
   - 开关类设置使用 Switch 组件
   - 选择类设置使用 Select 组件
   - 输入类设置使用 Input 组件

6. **移除技术标记**:
   - 删除所有"T0XX"技术标记
   - 使用用户友好的标题和描述

## 为什么这么做

### 架构选择

1. **设置页分类**:
   - 通用设置:用户日常使用的配置
   - 模型配置:技术性配置,不常修改
   - 分类后降低认知负担

2. **从主界面移除配置**:
   - 主界面聚焦核心功能:语音输入
   - 配置项移到设置页,减少主界面复杂度
   - 符合用户心智模型:设置在设置页

3. **顶部水平 Tab 导航**:
   - 当前实现使用顶部水平 Tab
   - 为未来扩展预留空间(如"高级设置"、"关于")
   - 视觉层级清晰

### UI 设计

1. **设置页布局**:
   ```
   ┌─────────────────────────────────────────┐
   │  设置                                    │
   ├─────────────────────────────────────────┤
   │  通用 | 模型配置                         │
   ├─────────────────────────────────────────┤
   │        设置内容区                        │
   └─────────────────────────────────────────┘
   ```

2. **设置项组织**:
   - 使用卡片分组相关设置
   - 每个设置项:标题、说明、控件
   - 开关类设置:标题和说明在左,开关在右
   - 输入类设置:标题和说明在上,输入框在下

3. **保存按钮**:
   - 通用设置:统一的保存按钮在底部
   - 模型配置:每个模型独立的保存按钮
   - 保存成功后,toast 提示
