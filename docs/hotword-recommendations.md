# 热词推荐列表

基于测试结果，建议添加以下热词以提高识别准确率：

## 技术术语

### AI/ML 相关
- ASR（语音识别）
- OpenAI
- API
- GPT
- LLM

### 编程语言
- JavaScript
- TypeScript
- Python
- Rust
- React

### 编程概念
- async（异步）
- await（等待）
- Promise（承诺）
- try catch（异常捕获）
- finally（最终）

### 框架/工具
- Tauri
- GitHub
- Node.js
- npm
- Webpack

### 数据库
- MySQL
- MongoDB
- Redis
- PostgreSQL

## 使用方法

1. 进入应用的"热词"页面
2. 点击"添加热词"
3. 输入热词文本
4. 选择分类（如"技术术语"）
5. 启用热词

## 注意事项

- 热词会被注入到文本润色的上下文中
- 建议只添加常用的专业术语
- 过多热词可能影响性能
