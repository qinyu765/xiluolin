# UI 视觉风格

## 当前选择

项目当前采用 Notion 风格作为 UI 视觉参考。

选择原因：

- Notion 风格适合 productivity SaaS、knowledge management 和 workspace tools。
- XiLuoLin 的核心场景是办公、写作和编程输入，界面需要保持清晰、克制、可扫描。
- Notion 的 warm minimalism、soft surfaces 和高可读文本层级更适合当前桌面效率工具阶段。

## 应用范围

本阶段只应用基础视觉 token：

- 暖白背景。
- 白色工具卡片。
- 黑色主文本。
- Notion Blue 作为主操作和焦点色。
- 柔和边框和低强调度辅助背景。
- 仅在品牌标题处使用 serif fallback，正文仍使用系统无衬线字体。

## 不采用的内容

- 不做营销页式 hero。
- 不引入商品展示、品牌官网或目录页布局。
- 不使用大面积装饰图形、斜切块或复杂渐变。
- 不一次性重写所有组件视觉，后续按功能任务逐步补齐。
